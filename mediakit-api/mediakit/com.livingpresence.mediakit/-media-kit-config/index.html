MediaKitConfig

data class MediaKitConfig(val baseStreamUrl: String, val maxEventNumber: Int = DEFAULT_MAX_EVENT_NUMBER)

Configuration for the playback SDK. The base host is injectable so the hardcoded production server is no longer sprinkled through app code and the URL builder is unit-testable.

Parameters

baseStreamUrl

Absolute base that ends in /live/event, e.g. https://host:443/live/event. The event number and rendition suffix are appended, then /playlist.m3u8?DVR.

maxEventNumber

The largest event number to probe (inclusive). The production server currently serves events 1..20.
