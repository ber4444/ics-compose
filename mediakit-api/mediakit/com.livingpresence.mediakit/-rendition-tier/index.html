RenditionTier

enum RenditionTier : Enum<RenditionTier>

The unadvertised sibling renditions the Wowza server exposes per event, keyed by the URL suffix appended to live/event{i}. Measured 2026-07-03 against the production server:

tiersuffixresolutionbandwidthcodecs
P720(none)1280×720~1.0 Mbpsavc1.42c01f + mp4a
P360_360p640×360~507 kbpsavc1.4d401e + mp4a
P160_160p284×160~262 kbpsavc1.42c015 + mp4a
AUDIO_aacaudio-only~51 kbpsmp4a.40.2

The base master playlist advertises only P720; LadderSynthesizer turns the four siblings into a real ABR ladder client-side.
